Pretty Recipe Generator

Generates TeX files for recipes, which can be compiled into PDFs. An example, and free recipe, can be viewed here. Source code available here.

If including a picture, the file must be saved in the same folder as the .tex file with the same name as the .tex file. For best results, the picture should be a 1:1 aspect ratio.

Times suggested in the format 0h 00m

Input ingredients and directions one at a time and press add after each one.
If you want to input a temperature, do *F or *C

  • {{ingredient}} remove
  1. {{direction}} remove

Compiling

To compile into PDF, either upload on LaTeX Base, or locally with your own LaTeX distribution.

This Windows batch file can be used to convert multiple TeX files to PDF, if a LaTeX distribution is installed (Linux bash function available).
